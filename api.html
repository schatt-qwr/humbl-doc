API Reference

Complete API reference for the Humbl project. The project consists of four framework packages (packages/) and the main humbl_core Flutter plugin.

Framework Packages​

The four framework packages are documented in the LangChain Framework architecture page:

  • langchain_dart -- Core framework (Runnables, BaseTool, BaseMemory, BaseCallbackHandler, Embeddings, VectorStores)
  • langsmith_dart -- Observability (BaseTracer, RunEvaluator, Dataset)
  • litellm_dart -- Multi-provider gateway (Router, Providers, CostCalculator, SpendLog)
  • langchain_graph -- State machines (StateGraph, Channels, CheckpointSaver, ReAct agent)

humbl_core Sections​

Pipeline​

  • PipelineOrchestrator -- Entry point for running pipeline turns.
  • PipelineState -- Immutable state object that flows through every node.
  • StateGraph -- Generic graph engine with nodes and conditional edges.
  • Pipeline Nodes -- All 7 nodes and what each reads/writes.

Tool System​

  • HumblTool -- Abstract base class with five-gate security template.
  • ToolRegistry -- Registration, discovery, and bundle management.
  • Models -- Enums (AccessLevel, ToolGroup, ResourceType) and data classes (ToolResult, ToolContext).

LM Gateway​

  • ILmGateway -- Top-level gateway interface for LM routing.
  • Connectors -- ILmConnector interface and 11 built-in connectors.
  • Scheduling -- LmScheduler for priority preemption and background queuing.

Memory​

  • IMemoryService -- Unified memory abstraction (T2-T4).
  • ConversationStore -- Persistent conversation turn storage with quality scoring.
  • Vector Store -- IVectorStore and SqliteVecStore for semantic search.
  • EmbeddingGateway -- On-device + cloud embedding with fallback.

Devices SDK​

  • IPeripheralProvider -- Interface for adding new device types.
  • IConnectedDevice -- Connected device with capability interfaces.
  • DeviceRegistry -- Device discovery and management.

Security​

  • AccessControl -- Privilege math and gate enforcement.
  • PermissionService -- OS permission probing and tool state management.
  • ConfirmationService -- Multi-provider user confirmation.

Resources​

  • IHardwareResourceManager -- Lease-based hardware resource control.
  • ResourceLease -- Lease model, access modes, and events.

Services & Agent​

  • HumblAgent -- Always-free dispatcher with concurrent runs and scouts.
  • ServiceEventBus -- Typed pub/sub for inter-service communication.

Voice I/O​

  • VAD, STT, TTS -- Voice pipeline interfaces.
  • VoiceSessionRunner -- Main voice pipeline orchestrator.

Platform​

  • PlatformFactory -- Runtime platform selection.
  • Manager Interfaces -- All 25+ platform manager interfaces.
